# Crypto Dashboard Frontend
Frontend for the Crypto Dashboard project. This frontend gets built and packaged into the Custom Functions project automatically for the HarperDB Instance. Users will interact with this frontend to add new tracked cryptocurrencies, or add new feed handlers and categories.

![Preview](preview.png)

## Installation
```sh
# Install the required dependencies
yarn
# Edit the environment file to include your Instance URL
vim src/environments/environment.prod.ts
```
